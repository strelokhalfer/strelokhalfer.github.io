Методы

addLoadListener ( listner )

Цепляет функцию, которая будет вызвана как только изображение загрузится.

Параметры:

  • listner Функция

    Вызываемая функция

adjustTone ( r , g , b )

Задать оттенок всей области изображения.

Параметры:

  • r Number

    Величина красной составляющей (-255, 255)

  • g Number

    Величина зеленой составляющей (-255, 255)

  • b Number

    Величина синей составляющей (-255, 255)

blt ( source , sx , sy , sw , sh , dx , dy , [dw=sw] , [dh=sh] )

Переносит участок изображения и заполняет им принимающий участок.

Параметры:

  • source Bitmap

    Исходный Bitmap

  • sx Number

    Координата 'x' переносимого участка

  • sy Number

    Координата 'y' переносимого участка

  • sw Number

    Ширина переносимого участка

  • sh Number

    Высота переносимого участка

  • dx Number

    Координата 'x' принимающего участка

  • dy Number

    Координата 'y' принимающего участка

  • [dw=sw] Number [optional]

    Ширина принимающего участка

  • [dh=sh] Number [optional]

    Высота принимающего участка

blur ()

Применяет к изображению размытие.

clear ()

Очищает изображение.

clearRect ( x , y , width , height )

Очищает заданную область изображения.

Параметры:

  • x Number

    Координата 'x' очищаемого участка

  • y Number

    Координата 'y' очищаемого участка

  • width Number

    Ширина очищаемого участка

  • height Number

    Высота очищаемого участка

drawCircle ( x , y , radius , color )

Рисует круг.

Параметры:

  • x Number

    Координата 'x' центра круга

  • y Number

    Координата 'y' центра круга

  • radius Number

    Радиус круга

  • color Строка

    Цвет заливки в формате [#rrggbbaa]

drawText ( text , x , y , maxWidth , lineHeight , align )

Рисует текст с обводкой в Bitmap.

Параметры:

  • text Строка

    Текст

  • x Number

    Координата 'x' левой границы текста

  • y Number

    Координата 'y' верхней границы текста

  • maxWidth Number

    Ограничение текста по ширине

  • lineHeight

    Ограничение текста по высоте

  • align Строка

    Выравнивание текста (left, center, right и т.д.)

fillAll ( color )

Заливает Bitmap цветом.

Параметры:

  • color Строка

    Цвет заливки в формате [#rrggbbaa]

  • width Number

    Ширина заливаемого участка

  • height Number

  • color Строка

    Цвет заливки в формате [#rrggbbaa]

gradientFillRect ( x , y , width , height , color1 , color2 , vertical )

Заливает участок изображения градиентом.

Параметры:

  • x Number

    The x coordinate for the upper-left corner

  • y Number

    The y coordinate for the upper-left corner

  • width Number

    The width of the rectangle to clear

  • height Number

    The height of the rectangle to clear

  • color1 Строка

    The start color of the gradation

  • color2 Строка

    The end color of the gradation

  • vertical Boolean

    Whether it draws a vertical gradient

isError () Boolean

Checks whether a loading error has occurred.

Возвращает:

Boolean:
True if a loading error has occurred

isReady () Boolean

Checks whether the bitmap is ready to render.

Возвращает:

Boolean:
True if the bitmap is ready to render

Loads a image file and Возвращает a new bitmap object.

Параметры:

  • url Строка

    The image url of the texture

  • text Строка

    The text to be measured

Возвращает:

Number:
The width of the text in pixels

resize ( width , height )

Resizes the bitmap.

Параметры:

  • width Number

    The new width of the bitmap

  • height Number

    The new height of the bitmap

rotateHue ( offset )

Rotates the hue of the entire bitmap.

Параметры:

  • offset Number

    The hue offset in 360 degrees

Takes a snapshot of the game screen and Возвращает a new bitmap object.

Параметры:

  • stage PIXI.Stage

    The stage object

Возвращает:

Bitmap

Properties

baseTexture PIXI.BaseTexture

[только для чтения] The base texture that holds the image.

canvas HTMLCanvasElement

[только для чтения] The bitmap canvas.

context CanvasRenderingContext2D

[только для чтения] The 2d context of the bitmap canvas.

fontFace String

The face name of the font.

fontItalic Boolean

Whether the font is italic.

fontSize Number

The size of the font in pixels.

height Number

[только для чтения] The height of the bitmap.

outlineColor String

The color of the outline of the text in CSS format.

outlineWidth Number

The width of the outline of the text.

paintOpacity Number

The opacity of the drawing object in the range (0, 255).

rect Rectangle

[только для чтения] The rectangle of the bitmap.

smooth Boolean

Whether the smooth scaling is applied.

textColor String

The color of the text in CSS format.

url String

[только для чтения] The url of the image file.

width Number

[только для чтения] The width of the bitmap.
